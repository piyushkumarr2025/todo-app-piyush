# To‑Do Pro (Piyush)
Upgraded To‑Do app with **Priority, Categories, Sorting, Dark/Light Mode, Stats** — built using HTML/CSS/JS with LocalStorage.

## Features
- Priorities (High/Medium/Low) + priority sort
- Category/label per task
- Due date with overdue highlight + Today/Overdue stats
- Sorting: Created, Due, Priority, Title
- Filters: All / Active / Completed / Overdue
- Search box
- Dark/Light mode (saved)
- Export/Import JSON
- Clear Completed
- Responsive mobile-friendly UI

## How to update your existing GitHub Pages
1. Download ZIP and extract.
2. In your repository (`todo-app-piyush`), click **Add file → Upload files**.
3. Replace existing `index.html`, `style.css`, and `app.js` with these upgraded ones.
4. Commit changes → The site auto-updates within ~1 minute.
