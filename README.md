# To‑Do List Web App (Piyush)
Simple, mobile‑friendly To‑Do app using **HTML + CSS + JavaScript** with **Local Storage**.

## Features
- Add, complete, edit, delete tasks
- Optional due date
- Filters: All / Active / Completed
- Search
- Clear completed
- Export / Import JSON
- Works offline (data in localStorage)
- Mobile friendly UI

## How to run (Phone or Laptop)
1. Extract the ZIP.
2. Open `index.html` in your browser (Chrome recommended).
3. That’s it.

### Mobile (Android) Tips
- Install **Acode** or **Spck Editor** to edit files.
- You can also upload the folder to **Replit** or **GitHub Pages** to show it on any laptop.
